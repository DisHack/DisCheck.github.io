Настройка WR-Форума

:: Назначение скрипта ::
Перед Вами самый обычный форум. Хотя, наверно, не самый обычный. Дело в том, что этот скрипт - ОЧЕНЬ маленький форум. Размер: исполняемого файла - 10,0 Кб; всего форума - 23 Кб (с учётом смайлов, и хэлпа README.html).
:: Возможности ::
1. Возможность выбора своего заголовка для всех страниц форума (увеличивает рейтинг форума в поисковых системах). Сделать это можно заменив значение переменной $fname="WR-Форум на сайте wr.zgr.ru". (Будьте внимательны при создании заголовка и не используйте символы " / | \ )
2. Наличие девяти скинов для форума. Для установки скина закоментируйте текущую строку вида //$bdcolor="#FF9A00"; $fcolor="#FFFFFF"; $bagcolor="#FFE1B0"; // Оранжевый и раскоментируйте другую.
3. Автозамена смайликов на картинки (в состав архива входят 11 классных смайликов: ).
:: Установка и использование ::
1. Для того чтобы пользоваться WR-форумом было удобно, создайте папку, например "wr-forum" (форум создаёт новые файлы и при большом количестве тем, найти какие-либо файлы будет сложнее). Обратите внимание на регистр букв, это имеет значения для верной работы скрипта рекомендую использовать нижний регистр.
2. Закачайте в эту папку все файлы, включая каталог со смайликами "smile". Закачивать и настраивать атрибуты скрипта можно менеджером "Total Commander".(Файл -> Изменить атрибуты).
3. Обратите внимание на обязательный состав форума:
  • index.php - Главная страница;

  • style.css - таблица стилей;

  • README.html - этот файл;

  • forum.dat - список тем ("хранилище" всех тем форума);

  • 21112136.dat - файл с сообщениями одной темы (цифры могут быть другие).

  • папка "smile" с 11-ю файлами внутри с расширением "gif".

    • 4. Установите всем файлам атрибуты Chmod 644 (для бесплатного хостинга на h1, h10, h11 и др.). При другом хостинге - 777.
    5. Запустить скрипт можно ссылкой такого типа: http://Мой_сайт/wr-forum/index.php
    6. Внимание! Скрипт написан на PHP и не работает на бесплатном хостинге НЕ поддерживающего эту язык, например, www.narod.ru.
    7. Если все пункты вы выполнили, а скрипт так и не запустился, ну чтож, пишите письмо на моё мыло, помогу чем смогу.
    З.Ы. Если у кого есть идеи по созданию интересного проекта в интернете, пишите. Есть хостинг http://wr.zgr.ru, возможность и желание.
